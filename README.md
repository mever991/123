Функционал

Базовый (обязательный)
**Добавление задачи** — введите текст и нажмите кнопку или клавишу Enter.
**Отображение списка** — все задачи выводятся в виде списка.
**Отметка о выполнении** — клик по чекбоксу переключает статус задачи (выполнена/не выполнена).
**Удаление задачи** — кнопка с крестиком удаляет задачу из списка.

Дополнительный (реализован)
**Сохранение в LocalStorage** — задачи не исчезают после перезагрузки страницы.
**Фильтры** — показывать: все задачи / только активные / только завершённые.
**Адаптивная вёрстка** — корректно отображается на мобильных устройствах и планшетах.
**Счётчик задач** — отображается общее количество задач в заголовке.
**Информационная панель** — показывает количество активных и завершённых задач.
**Очистка завершённых** — массовое удаление всех выполненных задач.

Как запустить

### Локальный запуск
1. Скачайте все три файла проекта:
   - `index.html`
   - `style.css`
   - `script.js`
2. Поместите их в одну папку.
3. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).
